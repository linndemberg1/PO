# Algoritmo Simplex e Análise de Sensibilidade

Este projeto implementa o Método Simplex em Java para resolver um problema de Programação Linear de maximização. A solução utiliza a estrutura de tableau simplex, realizando automaticamente as etapas de escolha das variáveis pivô, pivoteamento e atualização da base até encontrar a solução ótima.

Além da resolução do problema primal, o projeto calcula o problema dual por meio dos preços sombra e realiza análise de sensibilidade, avaliando o impacto de variações nos parâmetros das restrições sem alterar a base ótima.

## Tecnologias Utilizadas

* Java
* Eclipse IDE
* Programação Linear
* Método Simplex
* Dualidade
* Análise de Sensibilidade
