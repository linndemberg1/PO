Este projeto implementa o Algoritmo Simplex em Java para resolver problemas de Programação Linear de maximização. 
A solução foi desenvolvida utilizando a estrutura de tableau simplex, realizando automaticamente as etapas de seleção das variáveis pivô, 
pivoteamento e atualização da base até encontrar a solução ótima. 

Além disso, o projeto calcula o problema dual por meio dos preços sombra e realiza análise de sensibilidade, 
avaliando a estabilidade da solução em relação a variações nos parâmetros das restrições. O objetivo é aplicar, na prática, 
os conceitos de Programação Linear, Método Simplex, dualidade e análise de sensibilidade.
