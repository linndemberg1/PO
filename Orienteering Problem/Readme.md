# Orienteering Problem (OP)

A resolução do Orienteering Problem (OP) está diretamente relacionada à proposta central do projeto, que consiste em modelar e resolver um problema de planejamento
de rotas para um robô em ambiente de resgate e exploração.

Nesse contexto, cada vértice representa um ponto de interesse, possivelmente associado a áreas com informação relevante ou “recompensas”, 
enquanto as arestas representam o custo de deslocamento entre esses pontos. O objetivo do algoritmo é selecionar o melhor subconjunto de vértices a serem visitados,
maximizando a recompensa total obtida sem ultrapassar a capacidade máxima de bateria disponível do robô.

Dessa forma, o OP é utilizado como uma abstração matemática do problema real, permitindo transformar decisões operacionais — como quais locais visitar e 
em que ordem — em um modelo otimizado. Isso possibilita um planejamento de missão mais eficiente, seguro e com melhor aproveitamento dos recursos energéticos.

Em resumo, o modelo do Orienteering Problem fornece a base teórica para representar e resolver o problema prático proposto no projeto final da disciplina.
